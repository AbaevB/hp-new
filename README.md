# Gulp-template-for-frontend-2024
Простая сборка Gulp для верстальщиков.

## Описание

В данной сборке Gulp  я решил объединить все удобные для фронтенд-разработчика опции, которые я видел у разных  авторов, но при этом не перегружать ее чем-то лишним и непонятным. Сборка сделана на актуальных на момент написания версиях **Node.js 20.18.0** и **Gulp 5.0.0**.

### Функуционал

1. Сборка *html* страниц из компонентов при помощи *gulp-file-include*
2. Сборка *css* из *sass* или *scss*,  добавление вендорных префиксов, sourcemap, минификация css для prod-версии.
